Welcome to my Dead By Daylight API!

Welcome to my

Dead By Daylight API!

Hello, fellow traverser of the fog! I couldn't find a reliable, up-to-date Dead By Daylight API and needed a project to work on, so I bring this offering. My goal was for this API to be a robust wealth of Dead By Daylight data for other developers to utilize in their own endeavors. I will do my best to update data as it is adjusted per update, but alas, I am human and may miss some things. Some TODOs include: items, offerings, and add-ons. If you need any further explanation on how this API works, please refer to this README. Otherwise, if you have any comments, feedback, or you just want to share something you built while using this API, please feel free to reach out to me here. All rights to any and all of this information belong to Behaviour Interactive, and I do not claim to be the author of any of it. Thank you for stopping by and good luck in your next one!

Survivor Routes
/survivors

returns all survivors

/survivors

returns all survivors

/survivors/survivor_id

returns a specific survivor based on their unique id

/survivors/survivor_id

returns a survivor based on their unique id

/survivors/query?survivor_name=name

returns a specific survivor based on name query

/survivors/query?survivor_name=name

returns a survivor based on name query

Perk Routes
/perks

returns all perks

/perks

returns all perks

/perks/perk_id

returns a specific perk based on its unique id

/perks/perk_id

returns a perk based on its unique id

/perks/generic

returns all generic perks

/perks/generic

returns all generic perks

/perks/survivor

returns all survivor perks

/perks/survivor

returns all survivor perks

/perks/survivor/generic

returns all generic survivor perks

/perks/survivor/generic

returns all generic survivor perks

/perks/killer

returns all killer perks

/perks/killer

returns all killer perks

/perks/killer/generic

returns all generic killer perks

/perks/killer/generic

returns all generic killer perks

/character_id/perks

returns a specific character's perks based on the character's unique id
(survivor or killer)

/character_id/perks

returns a set of perks based on character's id

/perks/query?perk_name=perk_name

returns a specific perk based on perk name query

/perks/query?perk_name=perk_name

returns a perk based on name query

/perks/query?type=perk_type

returns all perks of that type
(supports queries: 'hex', 'boon', 'scourge', & 'teamwork')

/perks/query?type=perk_type

returns all perks of that type (e.g., 'hex')

/perks/query?character_name=character_name

returns all perks that belong to character of that name
(survivor or killer)

/perks/query?character_name=name

returns perks based on character's name

/perks/query?status_effect=name

returns all perks associated with specific status effect based on its name

/perks/query?status_effect=name

returns perks associated with status effect

/perks/query?status_effect=name&role=role

returns all perks associated with specific status effect based on its name, also specific to either 'killer' or 'survivor' role

/perks/query?status_effect=name&role=role

above, but specific to 'killer' or 'survivor' role
