Welcome to the Dead By Daylight API!

Survivor Routes
/survivors

returns all survivors

/survivors/survivor_id

returns a specific survivor based on their unique id

/survivors/query?survivor_name=name

returns a specific survivor based on name query

Killer Routes
/killers

returns all killers

/killers/killer_id

returns a specific killer based on their unique id

/killers/query?killer_name=killer_name

returns a specific killer based on name query

Perk Routes
/perks

returns all perks

/perks/perk_id

returns a specific perk based on its unique id

/perks/generic

returns all generic perks

/perks/survivor

returns all survivor perks

/perks/survivor/generic

returns all generic survivor perks

/perks/killer

returns all killer perks

/perks/killer/generic

returns all generic killer perks

/character_id/perks

returns a specific character's perks based on the character's unique id
(survivor or killer)

/perks/query?perk_name=perk_name

returns a specific perk based on perk name query

/perks/query?type=perk_type

returns all perks of that type
(supports queries: 'hex', 'boon', 'scourge', & 'teamwork')

/perks/query?character_name=character_name

returns all perks that belong to character of that name
(survivor or killer)

/perks/query?status_effect=name

returns all perks associated with specific status effect based on its name

/perks/query?status_effect=name&role=role

returns all perks associated with specific status effect based on its name, also specific to either 'killer' or 'survivor' role

Chapter Routes
/chapters

returns all chapters

/chapters/chapter_id

returns a specific chapter based on its unique id

/chapters/query?killer_name=killer_name

returns a specific chapter based on its associated killer's name

/chapters/query?survivor_name=name

returns a specific chapter based on an associated survivor's name

Realm Routes
/realms

/realms/realm_id

/realms/query?killer_name=killer_name

returns a specific realm based on its associated killer's name

Map Routes
/maps

returns all maps

/maps/map_id

returns a specific map based on its unique id

/realm_id/maps

returns maps in a specific realm based on unique realm id

/maps/query?realm_name=name

returns maps in specific realm based on realm's name

Power Routes

/powers/power_id

returns a specific power based on its unique id

/killer_id/power

/powers/query?killer_name=name

Weapon Routes

/weapons/weapon_id

Status Effect Routes
/status_effects

returns all status effects

/status_effects/status_effect_id

returns a specific status effect based on its unique id

/status_effects/buffs

returns all status effects that are buffs

/status_effects/debuffs

returns all status effects that are debuffs

/survivor/status_effects

returns all status effects that can apply to survivors

/survivor/status_effects/buffs

returns all status effect buffs that can apply to survivors

/survivor/status_effects/debuffs

returns all status effect debuffs that can apply to survivors

/killer/status_effects

returns all status effects that can apply to killers

/killer/status_effects/buffs

returns all status effect buffs that can apply to killers

/killer/status_effects/debuffs

returns all status effect debuffs that can apply to killers
