Welcome to the Dead By Daylight API!

Survivor Routes:

/api/survivors

returns all survivors

/api/survivors/:survivorId

returns a specific survivor based on their unique id

/api/perks

returns all perks

/api/perks/generic

returns all generic perks

/api/perks/survivor

returns all survivor perks

/api/perks/killer

returns all killers

/api/:characterId/perks

returns a specific character's perks based on the characters unique id

/api/perks/:perkId

returns a specific perk based on its unique id
